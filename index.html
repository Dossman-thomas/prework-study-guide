HTML

Hypertext Markup Language (HTML) is made up of different elements (wrapped in angled brackets, <" "> </" ">) that create the content and basic framework for all webpages by combining.

An element is the type of content on a webpage. Common examples of elements include:

  • Text elements
  • Image elements
  • Header elements
  • List elements

Elements are also used to divide and describe content on a page. examples of such elements include:

  • Meta-related elements
  • Content division elements
  • Semantic elements

The two most important elements for the structure of the webpage are the "head" and "body" elements.

  • The head element contains information, or metadata (unseen data about webpages that the browser uses to serve the webpage to users) about the webpage that is not visible to the user.
  • The body element contains the content that is visible to the user. All the content that we want the user to see and interact with are put inside this element.

NOTE: these two elements are examples of semantic HTML elements, or the elements that describes its conent and the role it plays on the page. It's important to use semantic elements whenever possible, because they make your HTML easier to read for other developers and the browser.

The "head" element

The head element may contain the following elements:

  • The 'charset' element sets the character encoding.
  • the 'http-equiv' element sets an HTTP header for conent.
  • The 'viewport' element sets the default view for our webpage.
  • The 'title' element defines the title of the webpage and is the only visible element in the head element.

The "body" element

The body element may contain other semantic elements including the following:

  • The 'header' element appears at the top of the page and usually contains a navigation menu, a title heading, and/or other introductory content.
  • 'heading' elements (e.g., h1, h2, h3, etc).
  • 'section' elements that divide the contents of the webpage into easier to read segments.
  • 'paragraph' elements (p) that contain text.
  • 'ul', 'ol', and 'li' elements represent unordered lists, ordered lists, and list items respectively.
  • A 'br' element creates a line break between blocks of content.
  • The 'a' element (stands for anchor) creates links to the same webpage or other webpages you provide the URL for using the 'href' attribute
  • The link element specifies relationships between the current document and an external resource. The 'src' attribute defines the location of the file. The 'alt' attribute contains a text string that describes the appeadance and functionality of the image. This is important for improving Search Enginge Optimization (SEO), screen readers, and other assistive technologies.
  • The 'footer' is last and appears at the bottom of the page. Usually, the footer element contains the author, copyright, contact, sitemap, and navigation of the webpage.

NOTE: an attribute either modifies the default functionality of an element type or provides functionality to certain element types unable to function correctly without them. Another example of important attributes are the "class" and "id" attributes. They help to name elements so they can later be targeted and used with CSS and JavaScript.
